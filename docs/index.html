Every few seconds, AutoLog screenshots your screen and extracts text via OCR. An LLM summarizes your activity into 5-minute chunks, infers what you're working on, and links related activities through shared files, URLs, and topics.

Features

Activity Knowledge Graph

App sessions, LLM-inferred activities, and cross-activity links via shared files, URLs, and topics. Not flat logs -- structured understanding.

Obsidian Integration

Activity notes named by what you did, not when. App stats, topic backlinks, daily dashboards -- all with [[wikilinks]] for graph exploration.

Privacy First

All data stays local. No cloud uploads, no telemetry. Password managers auto-excluded. LLM calls through your own proxy.

Built-in API

Local REST API at localhost:21890. Search summaries, browse activity, and integrate with Claude Code, MCP servers, or any tool.

Getting Started

Up and running in 2 minutes

1
Install
Homebrew cask or download the DMG from GitHub Releases.
2
Grant permissions
Screen Recording and Accessibility. The onboarding wizard walks you through it.
3
Connect an LLM
Point at a local claude -p proxy or paste an OpenRouter API key.
4
Done
Look for the eye icon in your menu bar. AutoLog is now capturing and summarizing.
