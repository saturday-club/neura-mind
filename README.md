# NeuraMind

The average person with ADHD loses hours to context-switching and executive dysfunction. NeuraMind is a macOS ambient memory engine that captures, connects, and resurfaces your work context so you never lose your train of thought again.
This project was built as part of the Claude Builder Club Hackathon @ IU on March 29, 2026.

## System Requirements

- macOS 26 Tahoe or later
- Apple Silicon Mac
- Screen Recording permission
- Accessibility permission
- Claude CLI installed and authenticated if you want LLM-powered features

## Install With Homebrew

```bash
brew tap dsatyam09/neuramind
brew install --cask neuramind
```

## Install From Source

```bash
git clone https://github.com/dsatyam09/NeuraMind.git
cd NeuraMind
make bundle
open .build/NeuraMind.app
```

## Privacy Disclosure

NeuraMind does not collect product analytics or upload your data on its own. Your data stays on your Mac unless you explicitly enable an external LLM workflow. If you use Claude-powered or other LLM-powered features, any external data handling depends on your provider and your own opt-out or privacy settings, including your Claude settings.
